# The Eddie information-hub sample — image credits

All photos below are license-clean (public-domain or Creative-Commons) images
sourced from Wikimedia Commons, and each license was verified via the Commons
API before use. They are used only as sample scenery placeholders in the BlueWave
Projects information-hub sample; a real build would use the organization's own or
licensed photography. Credits are also surfaced on-page (on-image credit pills +
the honest sample note).

Every image here is license-clean Waimea Bay / North Shore / big-wave **scenery**.
NONE depicts Eddie Aikau himself, the actual Eddie Aikau Big Wave Invitational,
its competitors, sponsor (Rip Curl / Quiksilver) branding, or the Aikau family —
those slots deliberately keep the reverent designed SVG art.

## Notes
- The two North Shore surf photos (Stan Shebs) and the North Shore surfer photo
  (André Fleckenstein) are generic North Shore / Oahu big-wave scenery — they are
  NOT represented as depicting a specific running of The Eddie or any competitor.
  The running-history cards state the year in text; the photo caption reads
  "North Shore · big-wave season" so the image is never mistaken for event footage.
- Searches for Commons imagery of "The Eddie" / "Eddie Aikau Big Wave
  Invitational" return many photos of the actual event and its competitors. Those
  were deliberately NOT used — the brief forbids event/competitor/portrait/branding
  imagery. Only license-clean Waimea / North Shore / big-wave scenery was selected.
