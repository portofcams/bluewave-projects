# Royal Coachman Lodge sample site — image credits

This is an unaffiliated cold-outreach sample built by BlueWave Projects. All
photos below are license-clean (public domain or Creative-Commons) images
sourced from Wikimedia Commons and verified via the Commons API
(`imageinfo` / `extmetadata`). None are Royal Coachman Lodge's own copyrighted
photography — the final build for the lodge would use their own official
photography, branding, and logo. Credits are also surfaced on-image and in the
page's honest sample-disclaimer footer.

These are generic Wood-Tikchik / Bristol Bay fly-fishing, floatplane, and
wildlife images — NOT photos taken at Royal Coachman Lodge itself, and NOT the
lodge's branding or logo. No identifiable guests are shown.

Notes:
- Originals were downloaded with curl (identifying UA + contact email) and
  converted to optimized WebP (`cwebp` q82; oversized originals downsampled
  with `sips` first to 1400-1600px on the long edge). Source JPGs were
  removed after conversion.
- A fifth candidate — "Sockeye salmon jumping over beaver dam, Lake
  Aleknagik" (Kristina Ramstad, CC BY-SA 3.0,
  https://commons.wikimedia.org/wiki/File:Sockeye_salmon_jumping_over_beaver_dam_Lake_Aleknagik,_AK_Kristina_Ramstad_1997.jpg)
  was found and license-verified but its only available resolution on
  Commons is 250x448px — too small for a clean web tile — so it was dropped
  in favor of designed SVG placeholder art for that slot.
- No fitting license-clean photo was found for the lodge's own
  cabins/accommodations, Scott Schumacher, or the specific Nuyakuk River
  stretch the lodge sits on — those sections keep designed SVG placeholder
  art with an honest "final build uses the lodge's own photography" note,
  consistent with every other tile that has no licensed photo behind it.
- These images depict Wood-Tikchik State Park, Lake Aleknagik, a
  De Havilland Beaver floatplane (different operator, Ward Air, Juneau), and
  a Wood River system creek — used as stand-ins for the region and aircraft
  type, not represented as taken at Royal Coachman Lodge's property or as
  the lodge's own aircraft.
